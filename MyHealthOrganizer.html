My Health Organizer App

For the development of this application, I researched health related issues impacting the elderly community. From the research I gathered, I come to the conclusion that simpler is better. I started off creating user journey maps in Mural. Next, I created a user flow chart of how the users will navigate throughout the app. After the journey maps and user flow chart were completed, I established a preliminary set of wireframes which were critiqued by a fellow UX designer. I brought all of these elements together to develop this health organizer app on mendix which is a low-coding application.
